# 🦕 Salón Jurásico

Sitio web desarrollado para un salón de fiestas infantiles temático, con una estética inspirada en dinosaurios y aventuras prehistóricas.

---

## 🚀 Tecnologías utilizadas

* ⚛️ React
* ⚡ Vite
* 🎨 Tailwind CSS
* 🧩 Componentes reutilizables
* 🔗 Wouter (routing)

---

## 📸 Funcionalidades

* Página de inicio atractiva
* Navegación entre secciones
* Galería de imágenes
* Página de contacto
* Diseño responsive (adaptado a mobile)

---

## 📦 Instalación y uso


Clonar el repositorio:

```bash
git clone https://github.com/TU-USUARIO/salon-jurasico.git
```

Entrar a la carpeta:

```bash
cd salon-jurasico
```

Instalar dependencias:

```bash
npm install
```

Ejecutar el proyecto:

```bash
npm run dev
```

Abrir en el navegador:

```
http://localhost:5173
```
